## 💻 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar meus conhecimentos em desenvolvimento backend.
Utilizei a linguagem Python junto à biblioteca Flask para entender melhor a criação de rotas, o processamento de formulários e a estrutura de uma aplicação web.
Também estudei a integração com banco de dados relacional utilizando MySQL, aplicando conceitos de SQL como criação de tabelas e execução de consultas (SELECT e INSERT).
No frontend, utilizei HTML e CSS para estruturar e estilizar as páginas.
O projeto foi realizado exclusivamente para fins de aprendizado e prática, com foco em fortalecer minha base em Flask e banco de dados.
